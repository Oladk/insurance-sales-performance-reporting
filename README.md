# Sales Performance Reporting – Insurance Business

## Context
In an insurance company, sales managers and commercial networks need clear, reliable
and actionable indicators to monitor performance, identify gaps versus objectives
and define corrective actions.

This project simulates an internal operational reporting system designed to support
sales performance monitoring in an insurance business (life insurance context).

## Business objectives
- Monitor sales performance through key commercial KPIs
- Track objectives vs actuals at advisor and unit level
- Identify underperforming areas and advisors
- Support managerial decision-making with clear monthly reports

## Key indicators (KPIs)
- Number of contracts
- Sales amount (premium / revenue)
- Conversion rate
- Performance vs objective (%)
- Time to close
- Advisor ranking (top / bottom)

## Target users
- Sales managers
- Network supervisors
- Commercial performance analysts

## Tools
- SQL
- Excel (Power Query)
- Power BI
- Python
